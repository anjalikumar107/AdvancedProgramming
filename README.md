# Androind Gerrit Team Health Dashboard

A dashboard that compares Android Software team projects on Gerrit.

## Scenario

This application is designed for a software engineering lead manager that needs to compare the health of multiple Android team projects hosted on Gerrit. Manual checking is time consuming as each project must be opened separately. This dashboard displays a central place where review health indicators like open changes, merged changes, abandoned changes, contributor activity, and backlog age can be viewed.

## Team Projects:

- Build Systems team: platform/build
- Core Systems team: platform/systems/core
- Frameworks team: platform/frameworks/base
- Settings team: platform/packages/apps/Settings
